**Swarms Tools** (`swarms-tools`) provides ready-made tool integrations you can attach to Python agents — finance data, web search, and social APIs — without writing schemas from scratch.

## Installation

```bash
pip install -U swarms-tools
```

Use alongside the core framework:

```bash
pip install -U swarms swarms-tools
```

## Basic usage pattern

```python
from swarms import Agent
# Import tools from swarms_tools for your vertical (see each guide)

agent = Agent(
    agent_name="Tool-Agent",
    model_name="claude-sonnet-4-6",
    tools=[],  # Add your swarms-tools callables here
    max_loops=3,
)

agent.run("Complete the task using available tools.")
```

<Tip>
  Tool functions need clear docstrings — Swarms converts them to OpenAI-style function schemas automatically.
</Tip>
