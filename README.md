# Swarms documentation

Documentation for **[Swarms](https://github.com/kyegomez/swarms)** — the multi-agent AI framework. This repo is the source for the docs site, built with **[Mintlify](https://mintlify.com)**.

## Run it locally

1. **Install the Mintlify CLI** (needs [Node.js](https://nodejs.org/)):

   ```bash
   npm i -g mint
   ```

2. **Clone this repo** and go to the root (where `docs.json` is):

   ```bash
   git clone https://github.com/The-Swarm-Corporation/swarms-framework-docs.git
   cd swarms-framework-docs
   ```

3. **Start the preview**:

   ```bash
   mint dev
   ```

4. Open **http://localhost:3000** in your browser.

If something fails, run `mint update` and try `mint dev` again.
