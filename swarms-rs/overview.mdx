## Overview

**swarms-rs** is a production-ready multi-agent orchestration framework built in Rust. It targets workloads that need low latency, fearless concurrency, and strong typing at scale.

## Key benefits

- **Multi-threaded architecture** — uses modern multi-core CPUs efficiently
- **Low overhead** — zero-cost abstractions and minimal runtime cost
- **Production-oriented** — suitable for real-time and high-throughput systems
- **MCP integration** — STDIO and SSE Model Context Protocol servers for tools

## When to use Rust vs Python Swarms

| Use swarms-rs | Use Python `swarms` |
| --- | --- |
| Latency-sensitive services | Fastest iteration and largest feature surface |
| Rust-native deployments | Jupyter, scripts, and most examples in this docs site |
| Embedded or systems teams | Full multi-agent catalog (GraphWorkflow, HeavySwarm, etc.) |
